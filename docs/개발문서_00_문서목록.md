# 실시간 Car Market 개발 문서 목록

## 문서 작성 목적

본 문서는 실시간 Car Market 프로젝트를 개발 산출물 형태로 정리하기 위해 작성했다. 요구사항 정의서에서 출발해 실제 구현 과정에서 확정된 기술 구조, 화면 흐름, API, 데이터 저장 방식, 검증 결과를 한곳에서 확인할 수 있도록 구성했다.

PDF 제출 시에는 아래 순서대로 묶으면 프로젝트의 기획 의도부터 구현 결과까지 자연스럽게 이어진다.

## 프로젝트 기준 정보

| 항목 | 내용 |
| --- | --- |
| 프로젝트명 | 실시간 Car Market 서비스 |
| 개발 형태 | React + Vite 프론트엔드, Node.js + Express 백엔드 |

## 문서 작성 기준

- 기존 요구사항 정의서와 실제 구현된 코드를 함께 기준으로 삼았다.
- 기능 설명은 제출자가 직접 설명할 수 있는 수준으로 풀어서 작성했다.
- 표와 흐름도는 전체 구조를 빠르게 파악하기 위한 용도로 추가했다.
- 환경변수 값, DB 접속 문자열, Firebase Secret 등 민감한 정보는 문서에 포함하지 않았다.
